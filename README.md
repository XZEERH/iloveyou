# 💖 Love Heart Animation Collection

Koleksi animasi **"I Love You"** berbasis Web (HTML/CSS/JS) dengan gaya Cyber-Neon. Project ini dibuat sebagai referensi bagi para developer yang ingin membuat efek visual romantis menggunakan matematika kurva parametrik.

> **Owner:** Razeerh 
> **Status:** Open Source / Referensi

---

## 🚀 Fitur Utama
* **Matematika Presisi:** Menggunakan rumus kurva hati untuk penempatan elemen yang sempurna.
* **GPU Optimized:** Animasi halus tanpa lag, bahkan di perangkat mobile.
* **Neon Aesthetic:** Tema gelap dengan pendaran neon pink yang modern.
* **Dual Version:** Tersedia versi *Rotating/Flow* dan versi *Blinking/Static*.

---

## 📂 Daftar Animasi

### 1. Animasi Love 1 (Rotating Flow)
Animasi ini membuat teks "I love you" mengalir seperti sabuk berjalan (conveyor belt) mengikuti alur bentuk hati secara terus-menerus.
* **Teknik:** `CSS offset-path` + `JavaScript Parametric Calculation`.
* **Karakteristik:** Dinamis, elegan, dan rapi.

### 2. Animasi Love 2 (Blinking Heart)
Teks "I love you" diam di posisi membentuk hati, namun muncul dan hilang secara bergantian (berkedip) dengan delay acak.
* **Teknik:** `CSS Keyframes` + `Opacity & Scale Transformation`.
* **Karakteristik:** Klasik, tenang, dan estetik.

---

## 🛠️ Cara Penggunaan
1. Pilih versi animasi yang ingin kamu gunakan.
2. Copy kode dari file `.html` yang tersedia.
3. Simpan dengan nama `index.html atau love.html`.
4. Buka di browser favoritmu atau jalankan melalui aplikasi editor seperti **Acode** atau **VS Code**.
