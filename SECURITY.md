# 🛡️ Security & Safe Use

Halo! Project **Love Heart** ini dibuat murni menggunakan HTML, CSS, dan JavaScript sederhana. Karena project ini tidak menyimpan data (no database) dan tidak meminta login, tingkat keamanannya sangat terjaga.

### 📜 Kebijakan Keamanan Santai
1. **No Data Collection:** Project ini tidak mengambil, menyimpan, atau mengirimkan data pribadi kamu ke mana pun. Semua berjalan di browser kamu sendiri.
2. **Safe Code:** Tidak ada skrip berbahaya atau pelacakan tersembunyi. Kamu bisa cek langsung kodenya di file `index.html`.
3. **Open Source:** Kalau kamu merasa ada kode yang kurang optimal atau bikin browser berat (lag), langsung aja kasih tahu ya!
